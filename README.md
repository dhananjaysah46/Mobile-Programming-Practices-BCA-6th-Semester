# Mobile Programming Practices (BCA 6th Semester)

This repository contains a comprehensive collection of Android application development practices and lab exercises completed during the 6th semester of the BCA program.

## 📌 Overview
The projects cover fundamental to advanced Android concepts using Java/Kotlin, ranging from basic UI components and Activity lifecycles to complex API integrations and database management.

## 📂 Project Directory

| Folder Name | Description |
|:--- |:--- |
| **ActivityLifecycleExample** | Demonstration of `onCreate()`, `onStart()`, `onPause()`, etc. |
| **ApiLogics / RetrofitExamples** | Networking and REST API integration logic. |
| **CrudExamples** | Database operations (Create, Read, Update, Delete). |
| **AdapterLogics / RecyclerLogics** | Implementation of Custom Adapters and RecyclerViews. |
| **FragmentLogics** | Static and Dynamic Fragment transactions. |
| **MapAppExample** | Google Maps API integration and location services. |
| **Question[1-6]** | Specific solutions to internal/external lab exam questions. |
| **AlertDialog / ToastDisplay** | UI feedback mechanisms and user interaction. |

## 🛠️ Tech Stack
* **IDE:** Android Studio
* **Language:** Java / Kotlin
* **Build System:** Gradle
* **Minimum SDK:** 21+

## 🚀 How to Run These Projects
1.  Clone the repository:
    ```bash
    git clone [https://github.com/dhananjaysah46/Mobile-Programming-Practices-BCA-6th-Semester.git](https://github.com/dhananjaysah46/Mobile-Programming-Practices-BCA-6th-Semester.git)
    ```
2.  Open **Android Studio**.
3.  Select **"Open an Existing Project"**.
4.  Navigate to a specific folder (e.g., `ActivityLogics`) and click **OK**.
5.  Wait for Gradle to sync, then run on an Emulator or Physical Device.
